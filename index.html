ROG Ally Gaming Guide

Quick reference for optimizing your gaming experience

Quick Decision Tree

  1. Does game support FSR 3?
    • Yes → Use FSR 3, disable AFMF2
    • No → Check base FPS
  2. Base FPS below 45?
    • Yes → Optimize base settings first
    • No → Consider AFMF2 if compatible

Path of Exile 2 Settings

AFMF2 currently causes issues with PoE 2
  • ✗ AFMF2: OFF
  • ✓ Anti-Lag 2: ON
  • Global Illumination: Medium
  • Shadow Quality: Medium
  • Dynamic Resolution: ON
  • Ambient Occlusion: OFF
  • Display Mode: Fullscreen
  • VSync: OFF

Memory Management

ROG Ally: 16GB Unified Memory

  • Keep ~8GB for system
  • ~8GB available for VRAM
  • Use medium textures if game needs >6GB VRAM

FSR 3 Setup

  • AFMF2: OFF
  • Anti-Aliasing: TAA
  • VSync: OFF
  • Frame Generation: ON
  • Quality Mode: Start with Balanced

Remember to restart game after enabling FSR 3

Quick Tips

  • Check AMD's FSR 3 game list first
  • Update drivers through Armory Crate
  • Disable shader cache with AFMF2
  • Monitor temps using AMD overlay
